## Overview

FreePeriod is an innovative smart vending system that provides accessible feminine hygiene products in public spaces, addressing a critical need for women's health accessibility.

## Technical Architecture

### Frontend Development
- Next.js 14 with App Router
- React 18 with TypeScript
- Tailwind CSS & shadcn/ui components
- Custom i18n for English-Chinese localization

### Location Services
- Google Maps API integration
- Real-time vending machine location tracking
- Proximity-based search functionality
- Interactive map interface

### Analytics Dashboard
- Recharts for data visualization
- Real-time inventory monitoring
- Usage pattern analysis
- Predictive restocking alerts

## Key Features

- **Smart Inventory Management**: Real-time tracking of product availability
- **Location-Based Services**: Find nearest vending machines
- **Multi-Language Support**: Seamless English-Chinese switching
- **Analytics Platform**: Usage insights and trend analysis
- **Mobile Responsive**: Optimized for all device sizes

## Business Impact

- HKUST(GZ) incubation program participant
- Addressing menstrual poverty in public spaces
- Improving accessibility to feminine hygiene products
- Data-driven insights for strategic placement
